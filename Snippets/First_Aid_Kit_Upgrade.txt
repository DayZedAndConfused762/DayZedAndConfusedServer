

****** install notes ******
First make sure you back up your files before adding any new code!

There is additional space in your First Aid Kits to add TetracyclineAntibiotics 
****************************************
Next to the  file name will be a comment,
New Entry or Replace Entry.
For New Entry - Place anywhere in the file (Better near the top).
For Replace Entry - Remove old code then place mine in.
****************************************

Upgrade your First Aid Kit by adding Tetracycline Tablets
****************************************
<!-- cfgspawnabletypes.xml New Entry  -->

<type name="FirstAidKit">
		<tag name="shelves" />
		<cargo chance="1">
			<item name="TetracyclineAntibiotics" />
		</cargo>
		<cargo chance="1">
			<item name="BandageDressing" />
		</cargo>
		<cargo chance="1">
			<item name="BloodTestKit" />
		</cargo>
		<cargo chance="1">
			<item name="BloodBagEmpty" />
		</cargo>

		</cargo>
	</type>
