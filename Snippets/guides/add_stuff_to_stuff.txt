****************************************
Next to the  file name will be a comment,
New Entry or Replace Entry.
For New Entry - Place anywhere in the file (I prefer near the top).
For Replace Entry - Remove old code then place mine in.
****************************************

example= <type name="Headtorch_Black"> will be the main item where <item name="Battery9V" /> will be the item added to the mail item so that when they spawn normaly they will come with this in it.
